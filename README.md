## <u>효과적인 회의 진행을 위한 실시간 피드백 및 회의 평가 시스템</u> Project Background
- ### 필요성
  - 기존 회의에서는 발언 내용 기록과 평가가 수동으로 이루어져 시간과 인력이 소모됨
  - 회의의 질과 효과성을 객관적으로 측정할 수 있는 지표가 부족함
  - 각 참여자의 기여도와 발언 품질을 정량적으로 파악하기 어려움
  - 회의 후 요약 및 보고서 작성에 추가적인 업무 부담 발생
  - 실시간 피드백 부재로 회의 중 개선이 어려움

- ### 기존 해결책의 문제점
  - 수동 회의록 작성은 시간이 많이 소요되며 누락 및 오류 발생 가능
  - 녹음 후 사후 분석만 가능한 기존 시스템은 실시간 개선 불가
  - 화자 구분이 되지 않아 참여자별 기여도 파악 어려움
  - 주관적 평가에 의존하여 객관성과 일관성 부족
  - 회의 데이터의 체계적 관리 및 분석 기능 미흡

## System Design
  - ### System Requirements
    - **실시간 음성 인식 시스템**
      - OpenAI Whisper Large v3 Turbo 모델을 활용한 한국어 음성 자동 전사
      - Silero VAD를 통한 정확한 음성 활동 감지
      - sounddevice 라이브러리 기반 실시간 마이크 입력 처리

    - **화자 자동 인식 시스템**
      - SpeechBrain ECAPA-VoxCeleb 임베딩 기반 다중 화자 구분 및 추적
      - 화자별 발언 시간 및 빈도 자동 추적

    - **AI 기반 발언 평가 시스템**
      - Ollama Gemma 3 270M 모델로 발언 내용 분석
      - 주제 일치성(0-10점) 및 신규성(0-10점) 자동 평가
      - LoRA 기법을 활용한 도메인 특화 파인튜닝

    - **실시간 대시보드**
      - React 18.3.1 + TypeScript 기반 웹 인터페이스
      - Recharts를 활용한 실시간 통계 시각화
      - WebSocket을 통한 실시간 데이터 동기화
      - Tailwind CSS + Radix UI 기반 반응형 디자인

    - **자동 보고서 생성 시스템**
      - AI 기반 회의 요약 자동 생성
      - reportlab을 활용한 PDF 보고서 생성
      - 화자별 통계, 주요 발언, 평가 점수 포함

    - **백엔드 API 서버**
      - FastAPI 0.120.1+ 기반 REST API 및 WebSocket 서버
      - PyTorch 2.9.0+ 딥러닝 프레임워크
      - Transformers 4.57.1+ 모델 로딩
      - 메모리 기반 세션 관리 (MeetingStatistics, SpeakerProfile)

    - **개발 및 배포 환경**
      - Python 3.12+ 백엔드 런타임
      - Vite 6.3.6 프론트엔드 빌드 도구
      - Ollama 로컬 LLM 서빙
      - 파일 시스템 기반 데이터 저장 (PDF, 설정, 학습 데이터셋)

## Case Study
  - ### Description
    **시나리오: 프로젝트 기획 회의**

    - **회의 주제**: 신규 모바일 앱 서비스 기획
    - **참여자**: 4명 (기획자, 개발자, 디자이너, 마케터)
    - **시스템 활용**:
      1. 회의 시작 시 각 참여자의 음성 샘플 등록
      2. 실시간으로 발언 내용 전사 및 화자 구분
      3. AI가 각 발언의 주제 일치성과 신규성 평가
      4. 대시보드에서 참여자별 발언 시간과 품질 점수 실시간 확인
      5. 회의 종료 후 자동 생성된 PDF 보고서 확인

    - **기대 결과**:
      - 회의 중 주제에서 벗어난 발언 감소
      - 참여도가 낮은 구성원의 발언 독려 가능
      - 회의록 작성 시간 90% 이상 절감
      - 객관적 데이터 기반 회의 문화 개선

## Conclusion
  - ### 프로젝트 성과
    - 회의 기록 및 평가 자동화로 업무 효율성 향상
    - 객관적 지표를 통한 회의 품질 개선 가능
    - 참여자별 기여도 정량화로 공정한 평가 기반 마련
    - 데이터 기반 의사결정으로 조직의 회의 문화 개선
    - 경량화로 서버 연결 없이 온디바이스 실행이 가능해 보안에 기여함

  - ### 기대 효과 및 활용 분야
    - 회의 내용의 체계적 아카이빙 및 지식 관리
    - 기업, 교육기관, 정부기관 등 다양한 분야로 확장 가능
    - 온라인 화상 회의 플랫폼과의 연동 가능성
    - 다국어 지원으로 글로벌 회의 지원 가능
